# React Counter App

A simple React counter application with a modern board-style UI.

## Features
- Increment, decrement, and reset the counter
- Modern dark/light board-style design
- Responsive and mobile-friendly
- Buttons with hover and active animations

## Technologies
- React (with Hooks)
- CSS (custom styles)
- Vite (for project setup)
